# Privacy Policy

## Save Image As Type Extension

This extension does not collect, store, or transmit any personal information.  
It only provides functionality to save images in different formats (PNG, JPG, WEBP) directly from the browser context menu.

- No cookies are used.  
- No analytics or tracking is performed.  
- No user data is shared with third parties.
